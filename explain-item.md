# shop管理系统项目说明

### 技术栈：

+ vue.js
+ vue-router插件实现路由监听
+ vue-cli 脚手架搭建项目
+ element-ui 组件库
+ axios 用来发送请求
+ 编程思想：组件化和模块化

### 登录实现

+ 使用路由导航守卫结合token处理视图访问拦截
+ 使用了element-UI的form表单组件来搭建了登录页面
+ 使用element-ui里的form组件中的表单自带验证方法实现基本的登录验证
+ 使用element-ui的messge消息提醒组件实现登录成功和失败的消息提醒
